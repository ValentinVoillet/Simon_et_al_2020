Project overview

RNA-seq and TCR-seq have been performed from patients having anti-PD1 treatment. Four fractions of CD8 T cells have been extracted (PD-1+TIGIT+, PD-1+, TIGIT+ and PD-1-TIGIT-) at different time points (T0, M1 and M2).

RNA-seq

  • Quality control

  • Exploratory analysis

  • Differential expression analysis
    • within each fraction - differences between time points
    • within each time point - differences between fractions
    • differences between outcomes (non-responders & responders)

TCR-seq

  • Quality control

  • Exploratory analysis
    • clonality and overlap sequences
    • clustering (T0, M1 & M2)
